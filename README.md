# 🚗 Vehicle Registration & Cryptography System

## 📌 Description
This project is a menu-driven console application developed in Python that demonstrates fundamental concepts of cryptography and blockchain. It includes SHA-256 hashing, digital signature generation and verification using RSA, and a vehicle registration system to securely store and retrieve vehicle details.

---

## 🎯 Features

### 🔐 SHA-256 Hashing
- Generate a secure SHA-256 hash for any input message

### ✍️ Digital Signature
- Generate public-private key pair
- Sign a message using the private key
- Verify the signature using the public key

### 🚗 Vehicle Registration System
- Register a vehicle with:
  - Number Plate
  - Owner Name
  - Model
- Retrieve vehicle details using number plate
- Prevent duplicate number plate entries

---

## 🛠️ Technologies Used
- Python
- hashlib (for SHA-256 hashing)
- cryptography library (for RSA digital signature)
